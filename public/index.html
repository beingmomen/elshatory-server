Authentication

Authentication Endpoints

Base URL: /api/v1/users

  • POST /api/v1/users/signup - Register a new user
  • POST /api/v1/users/login - Login with credentials
  • POST /api/v1/users/logout - Logout current user
  • POST /api/v1/users/forgotPassword - Request password reset
  • PATCH /api/v1/users/resetPassword/:token - Reset password with token
  • PATCH /api/v1/users/updateMyPassword - Update current user password
    Authentication required

Users

Users API

Base URL: /api/v1/users

  • GET /api/v1/users/me - Get current user profile
    Authentication required
  • PATCH /api/v1/users/updateMe - Update current user data
    Authentication required
  • DELETE /api/v1/users/deleteMe - Deactivate current user account
    Authentication required
  • GET /api/v1/users - Get all users
    Admin access required
  • GET /api/v1/users/admins - Get all admin users
    Admin access required
  • GET /api/v1/users/all - Get all users (no pagination)
    Admin access required
  • GET /api/v1/users/all/admins - Get all admin users (no pagination)
    Admin access required
  • POST /api/v1/users/admin - Create a new admin user
    Admin access required
  • GET /api/v1/users/:id - Get user by ID
    Admin access required
  • PATCH /api/v1/users/:id - Update user
    Admin access required
  • DELETE /api/v1/users/:id - Delete user
    Admin access required
